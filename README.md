Através de dados fictícios apresento um relatório de vendas com análise exploratória e painel visual (dashboard).


Objetivo do relatório:

 Analisar o desempenho de vendas por período e identificar padrões que auxiliem decisões comerciais.

Etapas da construção de meu trabalho:

Coleta dos dados:

Os dados foram gerados a partir de uma IA, adicione em um banco de dados SQLite para aprender  e praticar como conectar com  Python.

Analise Exploratória:

Para entender os dados a fundo usei um notebook  Pandas Python e extrai resultados significantes.

Painel Visual:

Para construção do dashboard interativo optei pelo Looker Studio.# relatorio_vendas
Através de dados fictícios apresento um relatório de vendas com análise exploratória e painel visual (dashboard).  
Os dados apresentam uma tabela pequena com 6 colunas e 51 linhas, sobre informações de vendas de cursos como Excel e Power Bi, assim também como formas de pagamento valores e datas.
A partir desses dados disponíveis, apesar de ser pouco, podemos tirar informações preciosas para a tomada de decisões futuras da empresa responsável.
